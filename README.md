# ocr
## 環境依賴
* node v12.20.0
* tesseract-ocr v4.0.x(https://github.com/tesseract-ocr/tesseract)

## 服務使用port
本專案使用 npm run serve時，除了啟動前端服務外，還會在3000 port啟動OCR接口。

npm install
